## データ定義
### セグメント
- 初来店：
当月初めての来店顧客
- リピート：
過去月に一度でも来店があり、かつロイヤルでも離脱でもない顧客
- ロイヤル：
直近３ヶ月間において月間総支払額がその月の平均を上回っており、かつ離脱でない顧客
- 離脱：
当月来店があり、次月来店のない顧客
